# Лабораторная работа N2 - Коллекция объектов

## Описание проекта

Реализация контейнерного класса ProductCatalog для хранения и управления объектами Product (товар) из лабораторной работы N1.

Цель работы: научиться работать с коллекциями объектов, реализовать собственный контейнерный класс с поддержкой итерации, индексации, сортировки и фильтрации.


### Скриншот N1 - Создание каталога и защита от дубликатов

![01](https://github.com/Raelag2/oop_labs/blob/main/images/lab02/4.jpg)

Что показано:
- Создание каталога с 4 товарами
- Вывод всех элементов через __str__()
- Защита от дубликатов - попытка добавить товар с существующим ID 101 отклонена

Проверяемые методы: add(), __str__(), __len__()

### Скриншот N2 - Итерация, поиск, индексация, удаление, сортировка

![02](https://github.com/Raelag2/oop_labs/blob/main/images/lab02/5.jpg)

Что показано:
- Итерация по каталогу (for item in catalog)
- Поиск товара по ID (find_by_id(103) -> "Монитор")
- Доступ по индексу (catalog[0], catalog[2])
- Удаление по индексу (remove_at(1) удалил "Мышь")
- Сортировка по цене (Клавиатура -> Ноутбук -> Монитор)
- Фильтрация (get_available() создал новую коллекцию из 3 товаров)
- Удаление по объекту (remove() удалил "Ноутбук")

Проверяемые методы: __iter__(), find_by_id(), __getitem__(), remove_at(), sort_by_price(), get_available(), remove()
